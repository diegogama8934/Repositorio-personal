Etiquetas multimedia.

Etiqueta <img>

La etiqueta <img> representa una imagen en nuestro documento HTML. Algunos de sus atributos son:

  • alt: Representa un texto alternativo que describe la imagen en caso de que la misma no pueda ser mostrada.
  • height:Define la altura de la imagen, se pueden poner el número específico de pixeles o en porcentaje.
  • src: Nombre de la imagen o su URL, básicamente sirve para insertar la imagen.
  • width: Define la anchura de la imagen, se puede poner el número específico de pixeles o en porcentaje.
  • title: Mostrará el contenido de este argumento cuando el usuario coloque el click sobre la imagen.

Ejemplo usando estas 5 propiedades:

Código usado para el ejemplo:

            
<img src = "../PaisajeUrbano.jpg alt="Imagen de paisaje urbano" width="30%" height="30%" title="Paisaje urbano">

Etiqueta <video>

La etiqueta <video> representa un video en el documento HTML. Algunos de sus atributos son:

  • src: Nombre del archivo de video.
  • controls: Indica si se deben mostrar los controles del reproductor.
  • height: Define la altura del video en pixeles o porcentajes.
  • loop: Indica si el video debe reproducirse una y otra vez.
  • poster: Especifica una imagen para mostrar en caso de que no se pueda reproducir el video.
  • title: Especifica una breve descripción del video que los navegadores y tecnologias de asistencia pueden mostrar.
  • width: Define la anchura del video en pixeles o porcentajes.

Ejemplo usando estas 7 propiedades:

Código usado para el ejemplo:

            
<video src = "../Video.mp4 controls height="30%" loop 
poster="../PaisajeUrbano.jpg" title="Calles de Nueva York" width="30%"></video>

Etiquetas para estructurar contenido.

Etiqueta <ul>

La etiqueta <ul> sirve para crear una lista desordenada (Unorder List). Los atributos de esta etiqueta van dirigidos a su diseño, en este caso mostraré solo 3:

  • disc: Representa la viñeta como un disco (un círculo relleno).
  • square: Representa la viñeta como un cuadrado.
  • circle: Representa la viñeta como un circulo (sin relleno a diferencia de disc)

Ejemplos usando las 3 propiedades:

  • Veracruz
  • Queretaro
  • Puebla
            
<ul style="list-style-type:disc">
    <li>Veracruz</li>
    <li>Queretaro</li>
    <li>Puebla</li>
</ul>
  • Oaxaca
  • Guanajuato
  • Michoacan
            
<ul style="list-style-type:square">
    <li>Oaxaca</li>
    <li>Guanajuato</li>
    <li>Michoacan</li>
</ul>
  • Sinaloa
  • CDMX
  • Nuevo León
            
<ul style="list-style-type:circle">
    <li>Sinaloa</li>
    <li>CDMX</li>
    <li>Nuevo León</li>
</ul>

Etiqueta <ol>

La etiqueta <ol> sirve para crear una lista ordenada (Order List). Usando el atributo type, podemos definir el símbolo que se mostrará en nuestra lista ordenada. A continuación se muestran las opciones para el atributo type:

  • 1: Números (viene por defecto) 1,2,3...
  • A: Letras mayúsculas A,B,C...
  • a: Letras minúsculas a,b,c
  • l: Números romanos en mayúsculas I,II,III...
  • i: Números romanos en minúsculas i,ii,iii...

Aunado a los atributos para el diseño del símbolo, existen otros 2 atributos que se pueden usar para otras finalidades:

  • start: Sirve para indicar a la lista un número de inicio con el cual empezará a contar la lista.
  • reversed: Para que los elementos se muestren de manera descendente.

Ejemplos usando los argumentos anteriores:

  1. Houston Rockers
  2. San Antonio Spurs
  3. Indiana Pacers
            
<ol type="1" start="4" reversed>
    <li>Houston Rockets</li>
    <li>San Antonio Spurs</li>
    <li>Indiana Pacers</li>
</ol>
  1. Sacramento Kings
  2. Memphis Grizzlies
  3. Miami Heat
            
<ol type="A" start="3" >
    <li>Sacramento Kings</li>
    <li>Memphis Grizzlies</li>
    <li>Miami Heat</li>
</ol>
  1. Los Angeles Lakers
  2. Los Angeles Clippers
  3. Golden State Warriors
            
<ol type="a" start="3" reversed>
    <li>Los Angeles Lakers</li>
    <li>Los Angeles Clippers</li>
    <li>Golden State Warriors</li>
</ol>
  1. Philadelfia 76´s
  2. Boston Celtics
  3. Brooklyn Nets
            
<ol type="I">
    <li>Philadelfia 76´s</li>
    <li>Boston Celtics</li>
    <li>Brooklyn Nets</li>
</ol>
  1. Chicago Bulls
  2. Detroit Pistons
  3. Toronto Raptors
            
<ol type="i" start="3" reversed>
    <li>Los Angeles Lakers</li>
    <li>Los Angeles Clippers</li>
    <li>Golden State Warriors</li>
</ol>

Etiqueta para enlace <a>

Esta etiqueta recibe el nombre de anchor y sirve para que al hacer click, el usuario pueda navegar a otra página web del especificada, ya sea una en general dentro de internet, o una del propio proyecto. Algunos de los atributos que ocupa son:

  • href: Contiene una URL o un fragmento de URL al cual apunta el enlace.
  • target: Especifica en donde desplegar la URL enlazada. Sus opciones son self, para que en la misma pestaña se cambie al enlace, o blank, para que el enlace se abra en una pestaña nueva.

Algunos ejemplos de estos argumentos:

  • Documentación de etiqueta anchor.
  • Página de la UAQ.

Etiqueta para tablas

Etiqueta <table>

Sirve para representar una tabla.

  • tr: Crea una fila en una tabla.
  • th: Crea el nombre de una columna de la tabla.
  • td: Crea una casilla de información en la tabla.

Etiqueta <colgroup>

Sirve para especificar informacion visual.

  • span: Sirve para decir cuántas columnas queremos afectar con dichos cambios.
  • style: Sirve para especificar la parte visual de la tabla.
  • Ambos argumentos se ponene dentro de la etiqueta col que se encuentra dentro de colgroup.

Etiqueta para formularios

Etiqueta <button>

Sirve para crear un boton. Algunos de sus argumentos son:

  • name: Asigna un nombre de control.
  • type: Indica el tipo de boton. Submit para configurar boton de envío. Reset para configurar un botón de reincio.
  • disabled: Indica que el control está desabilitado.

Etiqueta <input>

Input sirve para introducir datos. El argumento más conocido de input es type, sirve para definir qué tipo de dato ingresará.

  • text: Ingresa texto.
  • date: Ingresa una fecha.
  • color: Ingresa un color.
  • email: Ingresa un email.
